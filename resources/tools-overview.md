# Security Tools Overview

Security professionals rely on a variety of tools to analyze systems, monitor networks, and identify vulnerabilities.

The tools listed here are commonly referenced in cybersecurity learning environments.

## Network Analysis

Tools used to analyze network traffic and discover hosts or services.

Examples:

* Nmap
* Wireshark
* Netcat

These tools help security professionals understand network behavior and detect anomalies.

## Web Application Testing

Tools focused on analyzing web applications.

Examples:

* Burp Suite
* OWASP ZAP
* Web proxy analyzers

These tools assist in identifying misconfigurations and potential vulnerabilities.

## System Monitoring

Monitoring tools help detect suspicious activity in systems and networks.

Examples:

* Log monitoring platforms
* Intrusion detection systems
* Security event management tools

## Learning Note

Security tools should always be used responsibly and only on systems where authorization has been granted.

This repository documents concepts and study references rather than operational usage.
